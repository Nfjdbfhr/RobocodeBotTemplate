# Robocode Tank Royale Bot Template

A .NET template for creating bots for **Robocode Tank Royale** using C#.

This package installs a ready-to-run bot project with:
- A working example bot
- Cross-platform run scripts
- Proper Robocode metadata
- Customizable bot name via `dotnet new`

---

## Installation

```bash
dotnet new robocodebot -n MyAwesomeBot
